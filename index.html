Akka.js

Akka.js is a porting of Akka to the JavaScript runtime, meaning now Akka can run everywhere JavaScript can, including Node.js and browsers! Here are some demos ...

  • Raft Consensus Algorithm is a Raft implementation adapted from archie/raft, which showcases multicore computing through Web Workers
  • Reactive Calculator is an exercise from the Functional Reactive Programming course on Coursera that was ported to Akka.js to show possible ways of integrating Akka with the DOM
