Calculator

Once properly implemented, this calculator can compute values of variables depending on other variables.

Only very simple expressions are valid, here. It can be one of:

  • A number
  • The name of a variable (from 'a' to 'j')
  • An expression of the form '<expr> <op> <expr>' where <expr> is a number or a variable name, and <op> is one of + - * /
    Note that spaces are important, and that there cannot be more than one operation per cell.

Any error, such as a malformed expression, or cyclic references between variables, will result in NaN (Not-a-Number).
